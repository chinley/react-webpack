## 杂七杂八
 npm, nvm, nrm, npx, cnpm
 npm: 是 JavaScript 世界的包管理工具,并且是 Node.js 平台的默认包管理工具
 nvm: 不同node版本切换
 nrm: 切换npm源
 npx: 执行依赖包里的二进制文件
 cnpm:淘宝镜像
